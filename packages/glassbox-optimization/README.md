# glassbox-optimization

Optimization and hyperparameter search utilities under the `GlassBox.optimization` namespace, including components such as search helpers for model selection workflows.

## Installation

```bash
pip install glassbox-optimization
```

## Usage

```python
from GlassBox import optimization
from GlassBox.optimization.search import GridSearchCV
```
