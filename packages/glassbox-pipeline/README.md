# glassbox-pipeline

Pipeline construction and execution utilities under the `GlassBox.pipeline` namespace for chaining preprocessing and modeling steps in reusable workflows.

## Installation

```bash
pip install glassbox-pipeline
```

## Usage

```python
from GlassBox import pipeline
from GlassBox.pipeline.pipeline import Pipeline
```
