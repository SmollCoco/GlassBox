# glassbox-split

Data splitting utilities under the `GlassBox.split` namespace for train/test partitioning and related dataset split helpers.

## Installation

```bash
pip install glassbox-split
```

## Usage

```python
from GlassBox import split
from GlassBox.split.splitter import train_test_split
```
